## React Component for rendering text

A basic card based component to be used in profile pages as a container for each section.

### Component usage
```js
<ProfileSection title="Section title">
  This is the section content
</ProfileSection>
```

### Using props to customize the text

Modify the text to see it change live:
```js live
<ProfileSection title="Section title">
  This is the section content
</ProfileSection>
```
