## React Component for rendering text

A basic card to be used in profile pages as a container for each section.

### Component usage
```js
<ProfileCard title="Card title">
  This is the card content
</ProfileCard>
```

### Using props to customize the text

Modify the text to see it change live:
```js live
<ProfileCard title="Card title">
  This is the card content
</ProfileCard>
```
