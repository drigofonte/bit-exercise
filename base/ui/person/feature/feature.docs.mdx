## React Component for rendering text

A list item that shows a user's feature alongside an icon (e.g. age, hair colour, etc.)

### Component usage
```js
<Feature label="heading" value="feature value" icon={<div>a node</div>} />
```

### Using props to customize the text

Modify the text to see it change live:
```js live
<Feature label="heading" value="feature value" icon={<div>a node</div>} />
```
